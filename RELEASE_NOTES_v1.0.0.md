# gongzhonghao-peitu-fabu v1.0.0

首个可公开同步的版本。

## 这版包含什么

- 现成文章直入的公众号配图发布主链路
- `Gemini 生图 -> 去水印 -> bm-md 渲染 -> 草稿箱发布` 完整执行流程
- 技能内置 `gemini_driver.py`
- 技能内置 `remove_watermark.py`
- 内置水印模板资源
- `火焰编辑部老钱` 作为可选前置终审
- 中文 README、流程图、CHANGELOG、MIT 协议

## 这版解决了什么

- 不再依赖工作区外部脚本才能运行
- 不再默认混入写作逆向链路，职责更单纯
- 去水印流程被明确拆成独立阶段
- 发布粘贴优先走浏览器内 `Meta+V`，稳定性更高
- 登录态、环境文件、生成图和缓存默认不会被推到远程仓库

## 使用提醒

- 第一次使用前先跑登录初始化
- 生图一旦启动，不要中途打断
- 去水印之后再发草稿箱
- 上传前检查 `.env`、`.gemini/`、`assets/wechat_profile/` 是否被忽略
